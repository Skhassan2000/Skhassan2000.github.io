Bulk Product Upload

Upload multiple products at once using JSON or CSV format

Accepted File Formats

Please ensure your file follows one of these formats:

JSON Format Example:

[
  {
    "name": "Samsung French Door Refrigerator",
    "description": "Premium refrigerator with advanced cooling technology",
    "category": "Refrigerators",
    "image_path": "images/refrigerator.png",
    "price": 1999.99
  },
  {
    "name": "KitchenAid Professional Blender",
    "description": "High-powered blender for professional results",
    "category": "Small Appliances",
    "image_path": "images/blender.png",
    "price": 299.99
  }
]

CSV Format Example:

name,description,category,image_path,price
Samsung French Door Refrigerator,Premium refrigerator with advanced cooling technology,Refrigerators,images/refrigerator.png,1999.99
KitchenAid Professional Blender,High-powered blender for professional results,Small Appliances,images/blender.png,299.99

Notes:

  • Maximum file size: 10MB
  • Supported file types: JSON, CSV, TXT
  • Images should already be uploaded to the server
  • Required fields: name, category, price
